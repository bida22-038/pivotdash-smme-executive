# PivotDash MVP 1 for SMME Executive Intelligence

**PivotDash** aims to be a high-performance analytics engine designed for SMME Sales Executives who need instant insights without the complexity of Power BI.

### Key Features
- **Glassmorphism UI:** Modern, minimalist aesthetic optimized for executive focus.
- **Rule-Based ETL:** Automated cleaning of 500+ messy SMME sales records using Python/Pandas.
- **Executive PDF Reporting:** Capture dashboard states and notes into professional PDF reports.
- **Dynamic Charting:** Real-time chart type switching and theme toggling (Midnight vs. Frost).

### Tech Stack
- **Data Engine:** Python (Pandas/NumPy)
- **Frontend:** Vanilla JavaScript, Tailwind CSS, Chart.js
- **Reporting:** html2pdf.js
- **Environment:** Google Colab / Drive Integration

### How to Run
1. Download as a zip
2. Extract and open `index.html` in any modern browser.
